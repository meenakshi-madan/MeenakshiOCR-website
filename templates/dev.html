For fellow developers/students looking for more info about this project


Hello there! My name is Meenakshi Madan, and this app was developed as my third year (final year) project of B.Sc. I.T.

The project itself is an Android application as you probably already know, which performs Optical Character Recognition on images clicked with the camera OR selected from the gallery, displays the text, and lets the user do a bunch of things with this text.

I used the Tesseract OCR engine for the OCR part of it, but because the accuracy wasn't very good with a lot of images, I added an image processing library (imagemagick) to the app, figured out a sequence of filters that produced good accuracy by trial and error, and added that as well.

The entire UI is a 3D model made from scratch. The reason why I made the entire UI on blender (with very few edits in Gimp) is because of how real and awesome they make the UI look. It has a very robotic, sci-fi-ish feel to it, which is one of my favourite themes. Another reason of doing this is that edits become easy once you've made the basic model. You can change the texture of an icon in a few clicks, you can re-position a part of it and not have to worry about the 3D lighting, etc. I did edit them quite often so it was quite nice. Also, I like shiny things and blender knows how to make things shiny for me. :D

In the spirit of good open source code, I've decided to put my project up as well so that somewhere down the line, someone might benefit from some little part of my project. :)

The only problem is that since this is my final year project, I will only put up the source code (on bitbucket or github) and my Black Book (documentation) AFTER I have been evaluated and the marks are out, to prevent a situation where another student tries to show this project as their own. I've put a significant amount of time on this project and I don't want any un-pleasant side-effects. :I But as soon as I graduate, I will enable the links!
